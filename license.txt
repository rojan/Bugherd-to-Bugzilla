Bugherd to Bugzilla helps to import bugs from Bugherd to bugzilla.
It uses Bugherd php api and bz_webservice_demo.pl script which is available
in downloaded_bugzilla_folder/contrib/.
Find out more about bz_webservice_demo.pl in 
http://www.bugzilla.org/docs/tip/en/html/api/contrib/bz_webservice_demo.html
This program is tested on Bugzilla 4.2 and Bugherd api v1
